Run `cat /tools.txt` inside the container to view this file.

== NOTES ==
- Sandbox stays headless by design. If a tool requires clicking through a GUI, it is out of scope for solver lanes.
- Distfiles are mounted read-only at `/challenge/distfiles/`.
- Write repaired/generated files to `/challenge/workspace/`.
- Shared artifacts live under `/challenge/shared-artifacts/`.
- Services on the host are reachable via `host.docker.internal`.
- Wordlists live under `/opt/wordlists/`; use these before downloading ad-hoc lists during a solve.
- Run `sandbox-smoke-check` inside the container to verify major tooling after image updates.
